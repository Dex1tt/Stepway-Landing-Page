Лендинг-страница для вымышленного магазина кроссовок Stepway. Одностраничный сайт с каталогом моделей, формой обратной связи и адаптивной вёрсткой.

🔗 Демо: https://dex1tt.github.io/Stepway-Landing-Page/
Show Image

Возможности


🍔 Адаптивное меню-гамбургер для мобильных устройств
🎯 Активная подсветка пункта навигации при скролле (IntersectionObserver)
✨ Плавная fade-up анимация для hero-секции
👟 Каталог моделей с ценами
📩 Модальное окно с формой обратной связи
📱 Полностью адаптивная вёрстка под мобильные экраны


Технологии


HTML5 — семантическая структура
CSS3 — flexbox, @keyframes-анимации, медиа-запросы
JavaScript (Vanilla) — без фреймворков и библиотек
IntersectionObserver API — для подсветки активного раздела навигации


Структура страницы


Header — логотип, навигация, гамбургер-меню для мобильных
Hero — заголовок, описание, кнопка перехода к каталогу
Статистика — покупатели, количество моделей, рейтинг
Каталог моделей (#models) — карточки кроссовок с фото, названием и ценой
О нас (#features) — три ключевых преимущества бренда
Footer — логотип, навигация, ссылки на соцсети
Модальное окно — форма заявки, открывается по кнопке


Интересные технические решения


Активный пункт меню подсвечивается автоматически при скролле благодаря IntersectionObserver, без подсчёта координат секций вручную
Модальное окно закрывается кликом по фону, но не закрывается при клике внутри самой формы — реализовано через проверку event.target === modal, чтобы избежать всплытия события (event bubbling)
